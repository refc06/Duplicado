Herramienta Duplicados: Documentación
=====================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:

**«Duplicados»** es un sistema que permite identificar archivos duplicados en una ruta del disco duro del equipo donde es ejecutado. El mismo, extrae un *hash* de cada archivo en la ruta provista, y hace una comparaciòn de los datos. Este sistema *no busca duplicados de nombres*, sino de la información contenida dentro del archivo
